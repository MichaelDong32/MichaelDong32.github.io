Parentheses ( )

Parenthesis ( ) in JavaScript are used to hold contain parameters to be passed on to equations or for function calls, to surround conditional statements, or for grouping expressions to control the order of operation.

function myFunc() {
              if (condition1) {
              }

              if ( (1 + 2) * 3) {
                // very different from (1 + 2 * 3)
              }
            }
Curly braces { }

Braces { } are used during the declaration of Object Literals, or to enclose blocks of code (function definitions, conditional blocks, loops, etc).

var objLit = {
              a: "I am an Object Literal."
            };
Square brackets [ ]

Brackets [ ] are used to define Arrays and also used for accessing the properties of an Object by (or the elements of an Array) by it’s index, so mylist[2] fetches the third element in the Array.

var mylist = [1,2,3,4];
            console.log(mylist[2]);

            //returns the number 3
Single quotes ' '

Single quotes are used in enclose text to create strings

var  name = (‘mike’);
Double quotes " "

Double quotes can be used in the same way as single quotes for creating text strings. They are also used to define object properties when using object literal or bracket notation (Only double quotes are allowed in JSON).
